### 一、 标题栏设计改进描述 (Markdown)

#### 1. 视觉升级 (The 2025 Aesthetic)
*   **配色方案**：采用层次分明的深灰色系。标题栏背景色固定为 `#0c0c0e`，与主窗口的 `#09090b` 形成微弱对比，增加专业感。
*   **沉浸式体验**：完全移除系统默认边框，标题栏与侧边栏顶部无缝融合。
*   **精致图标**：放弃粗重的 `—` 和 `✕` 符号，改用 **Segoe Fluent Icons** 风格的细线图标（在代码中通过 Unicode 符号模拟），显得更轻盈。

#### 2. 交互与布局优化
*   **文字排版**：标题文字采用更现代的字体（如 Segoe UI Variable 或 Inter），字号减小至 11px，颜色使用中性的 `#71717a`，降低视觉重心，让用户关注聊天内容。
*   **按钮反馈**：
    *   **常规状态**：按钮背景透明，图标半透明。
    *   **悬浮（Hover）**：最小化/最大化按钮变为 `#27272a`（深灰）；关闭按钮变为经典的 `#e11d48`（胭脂红），图标变白。
*   **操作逻辑**：不仅支持拖拽移动窗口，还支持**双击标题栏最大化/还原**窗口，符合现代操作系统习惯。

### 二、 参考代码实现 (`TitleBar` 模块优化)

你可以直接用这段代码替换原 `main.qml` 中的 `titleBar` 部分。为了让效果更完美，我优化了窗口控制按钮的布局和交互逻辑。

```qml
// ------------------ 2025 现代沉浸式标题栏 ------------------
Rectangle {
    id: titleBar
    width: parent.width
    height: 32 // 减小高度，更显精致
    color: "#0c0c0e" // 略亮于背景的深锌色
    z: 100

    // 处理窗口拖拽移动
    DragHandler {
        onActiveChanged: if (active) window.startSystemMove()
    }

    // 双击标题栏：最大化 / 还原
    MouseArea {
        anchors.fill: parent
        onDoubleClicked: {
            if (window.visibility === Window.Maximized) 
                window.showNormal()
            else 
                window.showMaximized()
        }
    }

    RowLayout {
        anchors.fill: parent
        anchors.leftMargin: 16
        spacing: 0
        
        // 软件标题与小图标
        Row {
            spacing: 8
            Layout.alignment: Qt.AlignVCenter
            
            // 可选：添加一个极简Logo
            Rectangle {
                width: 12; height: 12; radius: 2
                color: "#3b82f6"
                anchors.verticalCenter: parent.verticalCenter
            }

            Text {
                text: "Digital Garden Chat"
                color: "#a1a1aa"
                font.pixelSize: 11
                font.weight: Font.Medium
                font.family: "Segoe UI Variable Display, Inter, sans-serif"
                renderType: Text.NativeRendering
            }
        }

        Item { Layout.fillWidth: true } // 弹簧占位

        // 窗口控制按钮组 (Windows 11 风格)
        Row {
            height: parent.height
            
            // 最小化按钮
            Rectangle {
                id: minBtn
                width: 46; height: 32
                color: minMouse.containsMouse ? "#27272a" : "transparent"
                Text {
                    anchors.centerIn: parent
                    text: "\uE921" // Fluent Icon 最小化
                    font.family: "Segoe Fluent Icons, Segoe MDL2 Assets"
                    font.pixelSize: 10
                    color: minMouse.containsMouse ? "white" : "#71717a"
                }
                MouseArea {
                    id: minMouse; anchors.fill: parent; hoverEnabled: true
                    onClicked: window.showMinimized()
                }
            }

            // 最大化/还原按钮
            Rectangle {
                id: maxBtn
                width: 46; height: 32
                color: maxMouse.containsMouse ? "#27272a" : "transparent"
                Text {
                    anchors.centerIn: parent
                    text: window.visibility === Window.Maximized ? "\uE923" : "\uE922" // 还原/最大化图标切换
                    font.family: "Segoe Fluent Icons, Segoe MDL2 Assets"
                    font.pixelSize: 10
                    color: maxMouse.containsMouse ? "white" : "#71717a"
                }
                MouseArea {
                    id: maxMouse; anchors.fill: parent; hoverEnabled: true
                    onClicked: {
                        if (window.visibility === Window.Maximized) window.showNormal()
                        else window.showMaximized()
                    }
                }
            }

            // 关闭按钮
            Rectangle {
                id: closeBtn
                width: 46; height: 32
                color: closeMouse.containsMouse ? "#e81123" : "transparent" // 标准关闭红
                Text {
                    anchors.centerIn: parent
                    text: "\uE8BB" // Fluent Icon 关闭
                    font.family: "Segoe Fluent Icons, Segoe MDL2 Assets"
                    font.pixelSize: 10
                    color: closeMouse.containsMouse ? "white" : "#71717a"
                }
                MouseArea {
                    id: closeMouse; anchors.fill: parent; hoverEnabled: true
                    onClicked: window.close()
                }
            }
        }
    }
}
```

### 三、 修改建议总结

1.  **高度微调**：将标题栏高度从 40px 降低到 **32px**。较窄的标题栏能让应用看起来更像一个工具而非网页，视觉上更精练。
2.  **字体渲染**：设置 `renderType: Text.NativeRendering`，这在 Windows 平台上能让小字号的标题文字显示得更锐利、清晰。
3.  **使用系统图标字体**：代码中使用了 `\uE921` 等 Unicode 编码，它们对应的就是 Windows 10/11 内置的 **Segoe Fluent Icons**。如果你的系统不是 Windows，可以简单地用常见的字符替换：
    *   最小化：`""` 或 `"—"`
    *   最大化：`""` 或 `"▢"`
    *   关闭：`""` 或 `"✕"`

这套修改方案能显著提升软件的“工业设计感”，使其更符合 digital-garden-chat 作为一个知识管理工具的调性。
